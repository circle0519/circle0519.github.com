近30年图形用户界面的发展编年史

2012年12月31日 | 评论

1973

1973年4月，首台可操作的 Alto 计算机在施乐 PARC 中完成。

1980

三江计算机公司推出 Perq 图形工作站

1981

1981年6月，施乐引入 Star 计算机，这是 Alto 商用机的继承者。

1983

1983年苹果公司的Lisa

Visi 公司发布的 Visi On，这是首个为 IBM PC 集成图形软件环境的系统

微软为 IBM PC 发布的 Windows 系统，但直到 1985 才推出

1984

苹果的 Macintosh

9月，数字研究公司公布其宝石图标/桌面用户界面和基于dos的电脑。后来被移植到Atari ST。

6月，window system X，来自麻省理工

1985

Geos 为 Commodore 64 和后来的苹果二代发布的系统

7月，Commodore 推出 Amiga 1000 ，包含 Amiga Workbench 1.0.

8月，微软最终发布的 Windows

1986

苹果公司威胁要起诉数字研究，因为GEM桌面看起来太像苹果的Macintosh。数字研究削弱桌面应用程序所以苹果不会起诉。

Tandy 为他们的 Tandy 1000 EX 发布了个人 DeskMate

1987

Apple Macintosh II, 首个彩色的 Macintosh.

Windows 2.03

Acorn 为 Acorn 计算机发布 “Arthur”，RISC OS 的基础

苹果发布 Finder 1.0 用于 Apple IIGS ProDOS 16 v1.3 (后来改为 GS/OS) 操作系统。

1988

IBM 发布 OS/2 1.10 ，这是微软开发的，很像 Windows 2

10月，NeXT 计算机发布，售价 6500 美金，包括 25 MHz '30 处理器, 8 MB RAM, 250 MB 光盘驱动，数学处理器，实时声音的数字处理器，传真调制解调器和 17 寸的监视器。

1990

Commodore 为 A3000 发布 Amiga Workbench 2

Windows 3.0

GeoWorks 发布了 PC-GEOS

1992

IBM 发布 OS/2 2.0 版本，一个真正的 32 位的操作系统

3月微软发布 Windows 3.1

Quarterdeck Office Systems 发布 Desqview/X

9月 Amiga Workbench 3 发布，用于 AGA Amigas.

1994

QNX 软件系统公司发布了首个可嵌入式的微内核系统 —— Photon microGUI

1995

Windows 95 来了！！！

BeOS at Agenda 96

1996

New Deal 发布 New Deal Office 2.5, 前身是 PC-GEOS.

IBM 发布 OS/2 Warp 4

Windows NT 4.0，界面跟 Windows 95 一样

1997

Mac OS 8 发布，在这个期间内销售成绩最佳

1998

6月25日，微软发布 Windows 98

7月12日，KDE 1.0 发布

11月22日 Shane Brooks 发布 98Lite, 这是一个删除了 IE 浏览器的 Windows 98

1999

3月3日，GNOME 1.0 发布

3月，苹果发布 Mac OS X 服务器版

6月，RISCOS Ltd 发布 RISC OS 4 for RiscPC, A7000 or A7000+ 机器.

2000

1月5日，苹果发布 Aqua，包含一个全新外观的 MacOS X client.

2月17日，微软发布 Windows 2000

2001

10月15日，微软发布 Windows XP（至今仍有不少人在使用）

2003

4月24日微软发布 Windows Server 2003

2004

1月 ReactOS 0.2.0 发布，这是一个 32位的 Windows 克隆，可使用 Windows NT 的设备驱动程序。

2007

1月30日，微软发布了 Vista 系统

2009

7月微软发布 Windows 7

2011

4月，Ubuntu 11.04 发布，使用 Unity 桌面系统

4月 Gnome 3 发布

8月19日，MATE 桌面系统发布，Fork 自 Gnome 2

2012

8月微软发布 Windows 8
